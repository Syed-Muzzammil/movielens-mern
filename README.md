# 🎬 MovieLens MERN App

A full-stack movie web application built with the MERN stack (MongoDB, Express, React, Node.js). Users can view trailers, search movies by actor/title, rate and review them, and more!

## 🔗 Live Demo

👉 [Click here to view the live website](https://movielens-mern.vercel.app/)

## 🛠️ Tech Stack

- 💻 **Frontend**: React, CSS
- 🌐 **Backend**: Node.js, Express.js
- 🗃️ **Database**: MongoDB Atlas
- 📦 **API Communication**: Axios
- 🔐 **Authentication**: JWT, bcrypt
- ☁️ **Deployment**: 
  - Frontend: [Vercel](https://vercel.com)
  - Backend: [Render](https://render.com)

## 🚀 Features

- 🔍 Search movies by title or actor
- 🎥 Embedded trailers
- ⭐ Submit ratings and reviews
- 📝 View other users' reviews and ratings


## ⚙️ Running Locally

```bash
# Clone the repo
git clone https://github.com/Syed-Muzzammil/movielens-mern.git
cd movielens-mern

# Frontend setup
cd frontend
npm install
npm start

# Backend setup (in separate terminal)
cd ../backend
npm install
node server.js
